<h2>Mac Address changer Utility for Windows</h2>
<li>
  <ul>Set Specific MAC address </ul>
  <ul>Set Random MAC address </ul>
</li>


<h2>Uses (Requires running in administrator)</h2>
<li>
  <ul>set specific MAC : MacEditor.exe 0f0f0f0f </ul>
  <ul>Set Random MAC: MacEditor.exe </ul>
  <ul>Set MAC for specific interface: MacEditor.exe -if="Interface1" </ul>
</li>
<h3>note: default interface is Ethernet !</h3>
